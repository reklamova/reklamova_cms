# Roadmap

## Etap 1 - Core skeleton

- struktura repo,
- bootstrap,
- admin `/admin`,
- health check,
- migracje,
- protected paths,
- dokumentacja update servera.

## Etap 2 - Installer

- webowy instalator,
- walidacja wymagan hostingu,
- konfiguracja bazy,
- tworzenie konta administratora,
- rejestracja site key.

## Etap 3 - Update ZIP

- generator paczek,
- podpis Ed25519,
- weryfikacja paczek,
- backup bazy i core,
- rollback,
- raporty update.

## Etap 4 - Central panel

- lista instalacji,
- status wersji,
- status hostingu,
- licencje,
- aktywne moduly,
- logi update.

## Etap 5 - Hostido pilot

- test na koncie wspoldzielonym,
- instalacja bez SSH,
- CRON albo web-cron,
- aktualizacja testowa z rollbackiem.
